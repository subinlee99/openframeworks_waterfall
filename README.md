**Week 2** 

1. ‘S’ key 를 눌러서 선택된 위치에서 물을 흐르게 한다. (Start)
    1.  물이지나는경로에선분이닿으면, 선분의두개의끝점중더낮은위치에있는쪽으로 물이 흐르도록 한다.
    2. 물이흐르는동안에는물이시작하는위치(선택된점)는바꿀수없다.
2. ’E’ key 를 눌러서 물을 멈추게 한다. (End)
3. ‘Q’key를눌러서동적할당된메모리해제및프로그램종료(Quit)
